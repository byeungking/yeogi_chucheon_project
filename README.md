# yeogi_chucheon_project

*프로젝트 설명
여기추천은 여행추천 사이트 입니다.  
추천 받기 버튼을 누르고 간단한 질문에 답을 하면, 질문에 맞춰 여행지를 추천해줍니다. 
여행지를 추천 받고 주변 숙박시설이나 레스토랑 등 추천 받을 수 있습니다.
